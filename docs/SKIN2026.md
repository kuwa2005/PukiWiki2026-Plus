# skin2026 — PukiWiki2026 Plus モダンスキン

PukiWiki2026 Plus が提供する **2026 向けモダンスキン**です。余白・タイポグラフィ・アクセシビリティを重視したレスポンシブ UI で、ライト／ダークテーマに対応します。

## ファイル配置

```
pukiwiki/skin2026/
├── pukiwiki.skin.php   … スキンテンプレート
├── pukiwiki.css        … スタイル（CSS 変数・ダークモード）
├── skin2026.js         … モバイルナビ・テーマ切替
├── main.js             … PukiWiki 標準クライアント
├── search2.js          … 検索 UI
├── ref-popup.js        … #ref ポップアップ
├── edit-dragdrop.js    … 編集 D&D（edit 時のみ読込）
├── .htaccess
└── index.html
```

既存の `pukiwiki/skin/` は変更しません。`skin2026/` は **独立ディレクトリ**として追加されます。

## 有効化手順

1. 本リポジトリを clone 済みであること（`pukiwiki/skin2026/` が存在すること）
2. `pukiwiki/pukiwiki.ini.php`（git 管理外）で `SKIN_DIR` と `SKIN_FILE` を設定:

```php
define('SKIN_DIR', 'pukiwiki/skin2026/');
define('SKIN_FILE', DATA_HOME . 'skin2026/pukiwiki.skin.php');
```

3. ブラウザで再読み込み（Ctrl+Shift+R）。DevTools Network で `pukiwiki/skin2026/pukiwiki.css` が **200** になることを確認

雛形は `pukiwiki/pukiwiki.ini.php.example` にコメント付きで記載しています。

## 設定の要点

| 定数 | 値 |
|------|-----|
| `SKIN_DIR` | `pukiwiki/skin2026/`（Web パス。末尾 `/` 必須） |
| `SKIN_FILE` | `DATA_HOME . 'skin2026/pukiwiki.skin.php'`（ファイルシステムパス） |

サブディレクトリ skin では **`SKIN_FILE` の指定が必須**です（[PUKIWIKI154-SKIN.md](https://github.com/kuwa2005/PukiWiki2026/blob/main/pukiwiki/docs/PUKIWIKI154-SKIN.md) 参照）。

## デザイン概要

- システムフォントスタック（日本語は Noto Sans JP / Hiragino 等にフォールバック）
- CSS カスタムプロパティによるライト／ダークテーマ（ヘッダーのトグルで切替、`localStorage` に保存）
- スティッキーヘッダー、カード型本文エリア、モバイル用ハンバーガーナビ
- 編集画面は従来スキン同様、ビューポート高さに合わせた textarea レイアウト
- PukiWiki プラグイン用クラス（表・カレンダー・検索ハイライト等）を継承
